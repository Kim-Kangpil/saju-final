=== 합충 실전 케이스 데이터베이스 ===
출처: 한국 명리학 커뮤니티 (2016-2024)

[C04] 육합 - 辰酉합
사주: 일간 甲, 일지 辰
발동: 2017 丁酉 세운 (酉가 일지 辰과 辰酉육합)
사건: 결혼 성사 (2017-10-31 실제 결혼식)
해석: 세운 지지가 일지와 육합을 이뤄 관성운 생성, 결혼 동기 강화
주의: 합으로 포장된 기신 중첩 가능성 → 길흉 단정 금지
신뢰도: 높음

[C05] 지지충 - 辰戌충
사주: 일지 辰
발동: 2018 戊戌 세운 (戌이 일지 辰을 충)
사건: 부부갈등 극대화 → 2019년 이혼 조정 착수
해석: 일지(배우자궁) 충 → 부부 불화, 관계 충돌, 2~3년 고비
신뢰도: 높음

[C06] 천간합 - 甲己합
사주: 일간 甲
발동: 2019 己亥 세운 (己가 일간 甲과 甲己합토)
사건: 이혼 조정 절차 착수 (2019-06)
해석: 천간합이 표면상 좋아 보이나 합화로 기신 강화 → 상황 악화
핵심교훈: 합이 반드시 길하지 않음, 합화 오행이 기신이면 오히려 흉
신뢰도: 높음

[C01] 삼합 - 亥卯未 목국
사주: 연주 乙亥, 월주 癸未, 일주 癸卯 (20대 남성)
발동: 원국 삼합 (대운 발동 아님)
사건: 조부 토지 개발로 100억대 자금 유입, 포르쉐 소유하면서도 요식업 현장 경험 중시
해석: 亥卯未 삼합 목국 → 식상(木) 극강, 조직보다 자영/기술축적/자기방식 고수 성향
특징: 강한 목 기운 = 돈보다 경험/성장 우선, 자유업 적합
신뢰도: 중간

[C08] 삼합 - 亥卯未 (대운+세운 발동)
사주: 연주 庚申, 월주 癸未, 일주 癸巳, 시주 乙卯
발동: 丁亥대운 + 2023 癸卯세운 (亥+未+卯 삼합 완성)
사건: 창의성 강화되나 사회적 활동/재정 규율 어려움 심화
해석: 대운 亥 + 세운 卯 + 원국 未 → 삼합으로 식상(木) 극강
핵심교훈: 삼합이 대운+세운으로 완성될 때 해당 오행 극강 발현
신뢰도: 중간

[C09] 천간충 - 丁癸충
사주: (동일 명식) 천간 丁火
발동: 2023 癸卯 세운 (癸 3개 + 丁 충거)
사건: 사회적 활동 저하, 관계/재정 압박
해석: 丁火(사회활동 연결 오행)가 癸水에 충거 → 목(식상)을 사회로 연결하는 불이 꺼짐
핵심교훈: 천간충으로 특정 오행 충거 시 해당 오행이 담당하던 삶의 영역 약화
신뢰도: 중간

[C10] 삼합 - 申子辰 수국
사주: 일간 戊, 지지 子+申 보유
발동: 壬辰 대운 진입 (辰이 합류해 삼합 완성)
사건: 선박 파산 회계 업무 시작 → 2년 내 10억+ 수익
해석: 戊土 일간 + 申子辰 삼합 수국 → 큰 물=큰 재물, 재성(水)이 극강해져 대형 재물운
핵심교훈: 삼합이 대운으로 완성될 때 해당 오행 관련 인생 전환점 발생
신뢰도: 중간

[C11] 천간합 - 丙辛합
사주: 일간 丙, 월간 丙(비견), 년간 辛(재성)
발동: 원국 구조 (상담 후 6개월 내 발현)
사건: 이직 후 파트너 협업으로 수수료 1억+ 달성
해석: 丙辛합 → 재물이 본인보다 동료/협업을 통해 먼저 들어오는 구조
핵심교훈: 천간합이 재성과 이뤄지면 단독보다 파트너십으로 재물 실현
신뢰도: 중간

[C02] 방합 - 寅卯辰 목방합
사주: 시주 壬寅, 일주 丁卯, 월주 丙辰, 연지 亥
발동: 원국 방합
사건: 소규모 사업체 근무, 재물 불안정
해석: 목 기운 과다 → 토·금 결핍, 현실 안정과 제어 부족
핵심교훈: 방합으로 특정 오행 과다 시 상극 오행 결핍으로 삶의 불균형 발생
신뢰도: 중간

[C03] 방합 - 巳午未 화방합 (붕충 포함)
사주: 연주 丁丑, 월주 丁未, 일주 戊午, 시주 丁巳
발동: 원국 방합 + 연지 丑과 월지 未의 丑未충
사건: 뮤지컬/노래 진로 고민, 가족 갈등, 독립 욕구
해석: 巳午未 방합 화 과다 + 丑未충이 방합 흔들기 → 정서 불안, 가정 마찰
핵심교훈: 방합이 충과 공존하면 방합 에너지가 불안정하게 발현
신뢰도: 중간

[C14] 천간충 - 丁癸충
사주: 시주 丁巳, 일주 癸巳, 월주 甲子 (의약계 종사자)
발동: 원국 천간충
사건: 관계 문제, 멘탈 불안 호소
해석: 丁(화)와 癸(수) 천간충 + 지지 화수 요동 → 심리 불안정, 감정 극단 대립
처방: 대운에서 토 기운 보강 시 대립 완화
신뢰도: 중간

=== 합충 해석 핵심 원칙 ===
1. 합이 반드시 길하지 않다 — 합화 오행이 기신이면 오히려 흉
2. 충이 반드시 흉하지 않다 — 기신을 충거하면 오히려 길
3. 삼합은 대운/세운으로 완성될 때 강하게 발동
4. 일지 충 = 배우자궁 불안정, 관계 변동
5. 천간충으로 특정 오행 충거 시 해당 오행 담당 영역 약화
6. 방합 + 충 공존 = 해당 방합 에너지 불안정 발현
